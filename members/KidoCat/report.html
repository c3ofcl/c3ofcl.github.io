2. 第2軸：じっくり作成モード (LLM + Image-to-Video)

じっくり作成モード 構成からこだわり、ブランドイメージを統一するためのプロフェッショナルな制作フローです。

Step 1: 広告構成と脚本作成 (LLM)

2026年の最先端モデルは、単なるテキストではなく、カメラワークまで指定した「ショットリスト」を出力します。

  • GPT-5.5 (OpenAI): 説得力のあるコピーライティングとエージェント機能に優れる 。
  • Claude 4.7 Opus (Anthropic): ブランドボイス（トーン）の維持能力が高く、洗練された脚本に強い 。
  • Gemini 3.1 Pro (Google): 100万トークンの文脈窓を活かし、長編のストーリー全体の一貫性を管理可能 。

Step 2: キービジュアル生成 (画像生成AI)

動画の「顔」となる高品質な画像を生成します。ここで「キャラクター固定」を行うことが重要です。

  • Midjourney V8.1: ネイティブHD 2K出力（2048x2048px）に対応。映画的な照明と質感において最高評価 。
  • Stable Diffusion 3.5: 特定の人物（生徒や教師など）をLoRA学習させ、完全に外見を固定した画像を生成可能 。

Step 3: 精密な動画化 (I2Vモデル)

生成した画像をベースに、指定した部分だけを動かします。

  • Runway Gen-4.5: 「Motion Brush」やカメラ制御が非常に強力で、広告業界のデファクトスタンダード 。
  • Kling 3.0 Omni: 5ヶ国語対応のリップシンクとマルチショット・ストーリーボード機能を搭載 。

3. 広告制作における「一貫性」維持のテクニック

キャラクター/商品DNAシステム :

AIは clip ごとに記憶をリセットするため、CHARACTER DNAという共通プロンプトブロック（名前、年齢、服装、特定のアザなど）をすべてのプロンプトに含める手法が推奨されます。これにより、カットが変わっても被写体が別人になることを防げます 。

4. 文化祭での実践的なコストシミュレーション

  • 低コストプラン (0円〜): 毎日付与される無料クレジットを活用。Kling 3.0 (毎日66クレジット) [7, 8]、Pika (デイリークレジット) [9]、Google Veo 3.1 (Google AI Studio経由) [9] を併用。
  • 高品質プラン (約7,000円〜): 準備期間の1ヶ月だけ Midjourney Standard ($30) と Runway Standard ($15) に加入し、高品質な素材を集中制作。
